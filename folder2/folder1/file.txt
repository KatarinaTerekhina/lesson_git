1. Посмотреть "где я" команда git: "pwd";
2. Создать папку команда git: "mkdir <название_папки>";
3. Зайти в папку команда git: "cd название папки";
4. Создание файлов команда git: "touch название файла с указанием расширения";
5. Показать содержимое папки команда git: "ls -la": 
                                           ls - выводит все файлы из списка
                                           la - отображает все скрытые файлы и каталоги;
6. Открыть файл команда git: "vim название файла с указанием расширения";
7. Открыть файл команда git: "notepad название файла с указанием расширения";
8. Выйти из папки на уровень выше команда git: "cd ..";
9. Инициализация/создание репозитория команда git: "git init";
10. Добавление файлов команда git: 1. Если добавляем конкретный файл : git add <имя_файла> 
                                   2. Добавить все файлы: git add . или git add --all
11. Добавление коммита команда git:git commit -m "<комментарий>"
12. Посмотреть историю изменений команда git:git log;
13. Показать статус файлов команда git:git status;
14. Показать объект в подробностях команда git:git show;
15. Показывает внесенные изменения команда git:git diff;
16. Показывает содержимое файла git: cat <имя_файла>;
17. Сделать запись (одновременно создать файл) git: echo "что вносим" >> <имя_файла>
