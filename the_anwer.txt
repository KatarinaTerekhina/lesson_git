1) Посмотреть где: я pwd
2) Создать папку: mkdir <name_folder>
3) Зайти в папку: cd <name_folder>
4) Создать файл: touch <name_file>
5) Вывести список содержимого папки: ls -la
6) Открыть любой txt файл: notepad <name_file.txt>
7) Выйти из папки на уровень выше: cd ..
—----------------------------------------------------------------------------------------------------------
8) Перемещение файлов в любую другую папку:  mv <name_file> <name_folder>/
9) скопировать любые 2 файла, которые вы создали, в любую другую папку: cp <name_file> <name_folder>/
10) Найти файл по имени -find, -name
11) Команда grep: используется для поиска любой строки или регулярного выражения
12) вывести несколько первых строк из текстового файла: head -n2 <name_file>
13) вывести несколько последних строк из текстового файла: tail -n2 <name_file>
14) Команда less: позволяет перемещаться по переданному файлу или куску текста, причём в обоих направлениях.
15) Вывести дату и время: data, git log --pretty='%ci | %s'
